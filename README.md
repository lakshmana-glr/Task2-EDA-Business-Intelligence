## Internship Task

This project is part of the **Data Analytics Internship at ApexPlanet Software Pvt. Ltd.**

**Task 2:** Exploratory Data Analysis (EDA) & Business Intelligence

The objective of this task is to explore the dataset, identify patterns and trends, and answer important business questions using **Python, SQL, and data visualization**.

---

# Dataset

Dataset used in this project: **Titanic Dataset**

The dataset contains passenger information including:

* Passenger class
* Age
* Gender
* Fare
* Ticket information
* Port of embarkation
* Survival status

After completing **Task 1 (Data Cleaning & Wrangling)**, the cleaned dataset was used for further analysis.

# Exploratory Data Analysis (EDA)

EDA was performed to understand the structure of the dataset and uncover insights.

The following analyses were performed:

* Survival distribution of passengers
* Passenger class distribution
* Age distribution
* Survival rate by gender
* Survival rate by passenger class
* Fare analysis by passenger class
* Correlation analysis between numerical variables

Visualizations such as **histograms, bar charts, box plots, and heatmaps** were created to better understand the data.

---

# Business Questions Answered Using SQL

Several business questions were answered using SQL queries:

1. What is the total number of passengers?
2. What is the survival rate of passengers?
3. Which gender has the highest survival rate?
4. Which passenger class has the highest survival rate?
5. What is the average fare for each passenger class?
6. Which embarkation port has the highest number of passengers?

These queries help derive meaningful insights from the dataset.

---

# Key Insights

* Female passengers had a significantly higher survival rate compared to male passengers.
* First-class passengers had better survival chances than second and third-class passengers.
* Ticket fare varied significantly across passenger classes.
* The majority of passengers boarded from Southampton.
